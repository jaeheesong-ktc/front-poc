## Project Structure
모노 레포 환경의 프로젝트 구조

```text
front : ROOT Repository
├── apps : 앱 폴더
├── packages : 패키지 폴더
    ├── eslint-config : ESLint 설정 패키지
    ├── logo : 로고 패키지
    ├── typescript-config : typescript 설정 패키지
    ├── ui : 디자인 시스템 패키지 (진행중 🏗)
```
